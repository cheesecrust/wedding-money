# 축의금 관리 앱 — 분석 탭 요구사항

> 작성일: 2026-03-29  
> 상위 문서: `축의금_요구사항.md`

---

## 1. 개요

입력 탭에서 수집된 데이터를 기반으로 **차트 + 숫자 카드 혼합** 형태로 시각화하는 분석 탭.  
별도 서버 없이 LocalStorage 데이터를 실시간으로 읽어 렌더링.

---

## 2. 탭 구조

```
[ 입력 ]  [ 분석 ]          ← 탭 전환
```

- 탭 전환 시 차트 즉시 렌더링 (데이터 변경 감지)
- 입력 탭에서 데이터 추가/삭제 시 분석 탭 자동 반영

---

## 3. 섹션별 요구사항

### 3-1. 상단 요약 카드 (숫자)

총 4개 카드, 가로 1줄 배치.

| 카드 | 내용 | 비고 |
|------|------|------|
| 총 축의금 | 전체 합계 금액 | `₩` 포맷 |
| 총 인원 | 입력된 전체 인원 수 | `명` 단위 |
| 총 식권 | 전체 식권 합계 | `장` 단위 |
| 1인 평균 | 총 축의금 ÷ 총 인원 | `₩` 포맷 |

---

### 3-2. 관계별 축의금 합계 — 막대 차트

- **차트 종류**: 세로 막대 차트 (Bar Chart)
- **X축**: 관계 (친구 / 직장동료 / 친척 / 선후배 / 기타)
- **Y축**: 축의금 합계 (원)
- **각 막대 위**: 금액 레이블 표시 (예: `150,000원`)
- **관계 미입력** 데이터는 "기타"로 집계
- 차트 우측 또는 하단에 관계별 숫자 테이블 함께 표시:

| 관계 | 인원 | 합계 | 비율 |
|------|------|------|------|
| 친구 | 5명 | 500,000원 | 45% |
| … | … | … | … |

---

### 3-3. 관계별 인원 비율 — 파이 차트

- **차트 종류**: 도넛 차트 (Doughnut) 또는 파이 차트
- **항목**: 관계별 인원 수 비율
- **범례**: 차트 우측 또는 하단 (관계명 + 인원 + %)
- 차트 중앙(도넛일 경우)에 **총 인원 수** 표시
- 컬러 매핑:

| 관계 | 색상 |
|------|------|
| 친구 | `#3B5998` (네이비 블루) |
| 직장동료 | `#5B8DB8` (스틸 블루) |
| 친척 | `#7FB3D3` (스카이 블루) |
| 선후배 | `#C9A84C` (골드) |
| 기타 | `#9E9E9E` (그레이) |

---

### 3-4. 식권 통계 — 숫자 카드 + 보조 차트

숫자 카드 3개 + 관계별 식권 막대 차트.

**숫자 카드:**

| 카드 | 내용 |
|------|------|
| 총 식권 수 | 전체 합계 |
| 식권 1인 평균 | 총 식권 ÷ 총 인원 (소수점 1자리) |
| 최다 식권 인원 | 식권 가장 많은 사람 이름 + 매수 |

**보조 차트:**
- 관계별 식권 합계 가로 막대 (Horizontal Bar)
- 각 항목 끝에 `N장` 레이블

---

### 3-5. 1인 평균 비교 — 관계별 가로 막대

- **차트 종류**: 가로 막대 차트 (Horizontal Bar)
- **내용**: 관계별 1인 평균 축의금
- **기준선**: 전체 평균 표시 (점선)
- 각 막대 끝에 금액 레이블

---

### 3-6. 관계별 축의금 랭킹 — 관계 그룹 순위표

관계별로 축의금 합계를 비교하여 어느 그룹이 가장 많이 냈는지 순위를 보여주는 섹션.

**표시 항목:**

| 순위 | 관계 | 인원 | 합계 | 1인 평균 | 식권 합계 |
|------|------|------|------|----------|----------|
| 🥇 1 | 친척 | 3명 | 450,000원 | 150,000원 | 6장 |
| 🥈 2 | 친구 | 5명 | 400,000원 | 80,000원 | 5장 |
| 🥉 3 | 직장동료 | 4명 | 200,000원 | 50,000원 | 4장 |
| 4 | 선후배 | 2명 | 100,000원 | 50,000원 | 2장 |
| 5 | 기타 | 1명 | 50,000원 | 50,000원 | 1장 |

**상세 규칙:**
- 축의금 합계 기준 내림차순 정렬
- 관계 미입력 데이터는 "기타"로 집계
- 1~3위: 메달 이모지 + 강조 배경 스타일
- 동점 처리: 합계 동일 시 인원 수 많은 쪽 우선
- 인원이 0명인 관계는 목록에서 제외
- 각 행 우측에 전체 대비 비율 표시 (예: `전체의 39%`)

---

### 3-7. 많이 낸 사람 Top 10 — 개인 랭킹 리스트

- 축의금 기준 내림차순 정렬, 상위 10명
- 데이터가 10명 미만이면 전원 표시

**각 항목 구성:**

```
[ 순위 ]  이름          관계     축의금
  🥇 1   홍길동        친구     200,000원
  🥈 2   이영희        친척     150,000원
  🥉 3   박민준        직장동료  100,000원
  4      김철수        친구      50,000원
  …
```

- 1~3위는 메달 이모지 + 강조 스타일
- 4위 이하는 일반 스타일
- 동점자 처리: 같은 금액이면 같은 순위 부여

## 4. 인터랙션

| 동작 | 설명 |
|------|------|
| 탭 전환 | 클릭 즉시 차트 렌더링 |
| 데이터 없음 | "아직 입력된 데이터가 없어요" 빈 상태 표시 |
| 차트 호버 | 툴팁으로 상세 수치 표시 |
| 내보내기 | 분석 탭 포함하여 공유용 HTML 내보내기 |

---

## 5. 기술 스택

| 항목 | 선택 |
|------|------|
| 차트 라이브러리 | Chart.js (CDN) |
| 데이터 소스 | LocalStorage (`chuwi_data`) |
| 렌더링 방식 | 탭 전환 시 `destroy()` 후 재생성 |
| 외부 의존성 | Chart.js 단일 CDN — 오프라인 시 번들 내장 필요 |

## 6. 레이아웃 구조

```
┌─────────────────────────────────────────────────┐
│  [ 요약 카드 × 4 ]                               │
├────────────────────┬────────────────────────────┤
│  관계별 축의금      │  관계별 인원 비율           │
│  막대 차트 + 표     │  파이/도넛 차트 + 범례      │
├────────────────────┴────────────────────────────┤
│  식권 통계 카드 × 3  +  관계별 식권 가로 막대    │
├─────────────────────────────────────────────────┤
│  1인 평균 비교 가로 막대 (전체 평균 기준선 포함)  │
├────────────────────┬────────────────────────────┤
│  관계별 축의금 랭킹 │  개인 Top 10 랭킹          │
│  (그룹 순위표)      │  (개인 순위 리스트)         │
└────────────────────┴────────────────────────────┘
```

## 7. 디자인 원칙

- UI 전체 컬러 시스템(`#993556` 핑크 팔레트)은 유지 — 버튼, 탭, 카드 테두리 등
- **차트 컬러만** 아래 별도 팔레트 적용 (핑크/빨강 계열 제외)
- 차트와 숫자 카드를 번갈아 배치 — 시각적 리듬감
- 모바일에서는 2컬럼 → 1컬럼으로 자동 전환
- 차트 배경 흰색, 테두리 `0.5px solid #e0e0e0`

### 차트 전용 컬러 팔레트

| 용도 | 색상 | HEX |
|------|------|-----|
| 1순위 / 친구 | 네이비 블루 | `#3B5998` |
| 2순위 / 직장동료 | 스틸 블루 | `#5B8DB8` |
| 3순위 / 친척 | 스카이 블루 | `#7FB3D3` |
| 4순위 / 선후배 | 골드 | `#C9A84C` |
| 5순위 / 기타 | 그레이 | `#9E9E9E` |
| 전체 평균 기준선 | 슬레이트 | `#607D8B` |
| 강조 (최고값) | 딥 네이비 | `#1A237E` |
